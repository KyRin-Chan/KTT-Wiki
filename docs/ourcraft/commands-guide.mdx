# 工艺小镇 指令说明

## 常用指令速查手册 📖

欢迎来到工艺小镇的指令速查手册喵！这里为你整理了服务器里超实用的指令，让你轻松上手，玩得更开心！(≧▽≦)

指令已经分好类别啦，你可以使用右侧的导航栏快速跳转到想看的部分哦。🔍

## 💬 聊天指令

| 指令 | 作用 |
| --- | --- |
| ` /msg <玩家名> <消息>` | 给指定玩家发送一条私密消息。📩 |
| ` /togglepm` | 开启或关闭你的私聊功能，以免被打扰。🔁 |
| ` /ignore <玩家名>` | 屏蔽某个玩家，你将不会再看到他的任何消息。🙉 |

## 🏠 领地指令

| 指令 | 作用 |
| --- | --- |
| ` /res info` | 显示你所在领地的详细信息。📝 |
| ` /res current` | 显示你所在领地的简要信息。👀 |
| ` /res list` | 列出你自己拥有的所有领地。📜 |
| ` /res unstuck` | 当你在领地里卡住时，将你移动到领地外。🚶‍♂️ |
| ` /res tpset` | 设置你当前所在领地的传送点（回家点）。📍 |
| ` /res setowner <新主人名>` | 将领地所有权转让给其他玩家。🎁 |

### 领地消息设置

你可以自定义玩家进出你领地时看到的消息哦！

| 指令 | 作用 |
| --- | --- |
| ` /res message <enter/leave> <消息>` | 设置进入或离开领地时的欢迎/告别语。✏️ |
| ` /res message remove <enter/leave>` | 移除进入或离开领地的消息。❌ |
| ` /res message default <参数>` | 使用预设的模板来重置消息。🔄 |

:::tip 提示
在设置消息时，你可以使用一些变量：
- `&1-9, a-f`：颜色代码，如 `&6你好` 会显示为 <span style={{color: "gold"}}>你好</span>。
- `%player`：会显示为进入/离开的玩家名。
- `%owner`：会显示为领地主人名。
- `%residence`：会显示为领地名。
:::

### 领地权限设置

使用 Flag（权限标签）来精细管理你领地的权限吧！

| 指令 | 作用 |
| --- | --- |
| ` /res set <权限> <true/false/remove>` | 为**所有玩家**设置整个领地的权限。 |
| ` /res pset <玩家名> <权限> <true/false/remove>` | 为**特定玩家**设置领地权限。 |

你也可以直接输入 `/res set` 或 `/res pset <玩家名>` 打开一个箱子菜单来进行图形化操作哦！📦

**常用权限标签 (Flag):**
- `build`: 建造/破坏权限
- `use`: 使用物品权限（如按钮、拉杆）
- `container`: 容器权限（如箱子、熔炉）
- `move`: 进入领地权限
- `tp`: 传送到领地权限

**举个栗子：**
- ` /res set build false`：禁止所有人在你的领地里建造。
- ` /res pset K2Lin build true`：只允许玩家 `K2Lin` 在你的领地里建造。

### 🎉 McMMO 组队

和朋友组队，可以共享 McMMO 经验，还不会互相伤害哦！

**队伍管理**
| 指令 | 作用 |
| --- | --- |
| ` /party create <队伍名> [密码]` | 创建一个新队伍。 |
| ` /party invite <玩家名>` | 邀请玩家加入你的队伍。💌 |
| ` /party accept` | 接受一个队伍邀请。 |
| ` /party kick <玩家名>` | 从队伍中踢出某个成员。😠 |
| ` /party disband` | 解散你的队伍。💥 |
| ` /party lock` / ` /party unlock` | 锁定/解锁你的队伍，决定是否允许他人自由加入。🔒 |
| ` /party password <密码>` | 为你的队伍设置或更改密码。🔑 |
| ` /party leader <玩家名>` | 将队长转让给其他成员。👑 |

**个人操作**
| 指令 | 作用 |
| --- | --- |
| ` /party` 或 ` /party info` | 查看你当前所在队伍的信息。👀 |
| ` /party join <队长名> [密码]` | 加入一个已存在的队伍。 |
| ` /party leave` | 离开你当前的队伍。😢 |
| ` /party help` | 查看更多队伍相关的指令。📚 |
