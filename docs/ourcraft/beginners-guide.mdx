## 基本规则 📜

为了让大家都能愉快玩耍，请遵守以下几条简单规则：

- **请勿在主世界挖矿** ⛏️
  - 需要资源？请前往**资源世界**（可通过跃迁中心到达）尽情挖掘！🌲
  - 资源世界会定期重置，不用担心资源枯竭。
- **请对其他玩家友好** 😺
  - 我们是一个和谐的社区，请互相尊重和帮助。禁止骚扰、辱骂、欺诈或偷窃。
  - 遇到冲突请联系管理员解决。
- **请勿破坏他人建筑** 💥

- **严禁任何形式的作弊** 🚫
  - 我们致力于维护公平的游戏环境。使用外挂、脚本等会被严厉处罚！

## 基本功能指南

### 🗺️ 主菜单与传送
- **打开主菜单**：潜行时切换副手 (`Shift+F`)。
- **快捷传送**：菜单内可进行随机传送、回主城、回家、去商店等操作。
- **跃迁中心**：通过主城的末地传送门可到达，是前往资源世界、竞技场、彩虹街等地的枢纽。系统商店就在彩虹街哦！🛍️

### 🚀 玩家间传送
想找朋友玩？使用以下指令：
- `/tpa <玩家名称>`：请求传送到该玩家位置。
- `/tpahere <玩家名称>`：请求该玩家传送到你的位置。

收到请求后，打开聊天框（`T`）点击 `[同意]` 或 `[拒绝]` 即可。

### 🔐 CraftLock 物品锁
用 `CraftLock` 保护你的箱子、门等方块！
- **锁定**：手持告示牌右键点击要锁定的方塊。
- **添加好友**：锁定后，在方块上再贴一个告示牌，就会出现 `[更多用户]` 告示牌。右键选中告示牌，输入 `/lock <行号> <用户名>` 即可添加好友权限。
- **临时解锁**：按住 `SHIFT`+右键点击可暂时允许他人使用。

## 经济系统入门 💰

服务器内使用一种名为 **OurCraft Token (OCT)** 的虚拟货币。

### 赚取与使用 OCT
- **赚取 OCT**：完成任务、出售物品、参与活动等。
- **使用 OCT**：购买物品、租用住宅、与其他玩家交易等。

### 查看余额
- **屏幕右侧**：动态显示你的钱包余额。
- **指令**：输入 `/balance` 查看余额，`/balancetop` 查看全服富豪榜。

### 交易方式
- **全球市场 (`/ah`)**：手持物品输入 `/ah sell <价格>` 上架出售。
- **QuickShop 商店**：
  1. 放置一个箱子。
  2. 手持想出售的物品，左键点击箱子创建商店。
  3. 在箱子中存货并设定价格。
  4. 去 Discord 或彩虹街公告板宣传你的小店吧！
- **玩家间转账**：使用 `/pay <玩家名称> <金额>` 直接给对方付款。
