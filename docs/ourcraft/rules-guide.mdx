# 工艺小镇 规则说明

## 欢迎来到工艺小镇！一起看看怎么玩才开心吧！

## 认识一下管理团队！

- **我们的工作人员和使命**🌟

    我们的使命是与大家一起玩得开心，创造美好的回忆喵~ 🥰

    为了让每位玩家都能享受愉快的游戏时光，我们制定了以下规则。👍

    如果你对规则有任何疑问，随时可以问我们的工作人员，他们会耐心为你解答的！🙋‍♂️

    我们的管理团队都是经过特别挑选和培训的人类（不是机器人哦！），他们会根据具体情况和证据，做出最公正的判断。👮‍♀️

    如果你对工作人员的处理有任何不满，或者觉得某个处罚/举报处理得不妥，请通过 [K2Lin Tickets](https://k2l.in/tk) 联系我们，我们会重新审查。📧

    同时，我们也会根据需要更新规则，所以大家要记得常来看看，确保自己了解最新的规则哦！📜

## 模组使用小贴士

- **来了解哪些模组可以用，哪些不可以吧！**🔧

    如果一个模组没有出现在下面的列表里，那就意味着暂时不能用哦。如果你希望我们把它加进来，可以[申请添加模组](https://k2l.in/tk)喵。🙋

    - **模组分类**📋
        我们根据模组对游戏体验的影响，把它们分成了4类。
        > **重要提示**：就算一个模组是允许的，也不能用它来作弊或者破坏游戏平衡哦！比如在黑暗迷宫里用夜视功能，这是绝对不行的！😠

    - **✅ 批准使用 (Approved)**
        这类模组主要提升游戏美感或提供便利，比如光影（Shaders）和物品整理（Inventory Tweaks），它们不影响公平性。😊

    - **⚠️ 有条件允许 (Conditional)**
        比如建筑投影（Schematica）和世界下载器（World Downloader）。它们本身无害，但在服务器里有一些使用限制，防止被滥用。😐

    - **🚫 禁止使用 (Forbidden)**
        任何能让你获得不公平优势的模组都严厉禁止！比如透视（X-Ray）、加速（Speed Hacks）、杀戮光环（Kill Aura）和鼠标宏等。😡

    - **❌ 无法兼容 (Incompatible)**
        这些模组在服务器上无法正常工作，通常是单机专用或需要我们没有安装的服务器端插件。😥

## 遇到坏蛋？举报他们！

- **如何举报违规的玩家喵？**👮‍♂️

    在游戏里看到有人骂人或作弊？你可以通过我们的 [K2Lin Ticket](https://k2l.in/tk) 系统来举报他们！🙋‍♀️

    虽然游戏里的工作人员会尽力帮忙，但他们可能正在忙别的事情。通过工单系统举报是最好的方式！

    创建一个**支持工单**，附上违规行为的截图证据📷，我们的工作人员就会尽快处理，有时甚至不需要登录游戏就能解决问题！👏

    > **请注意**：不要提交虚假或开玩笑的举报，滥用举报系统也会受到惩罚的哦！😠

## 服务器生存法则

    1. 保持OCN的纯净和友好
        - **言语文明**：不要说脏话或粗俗的话。🙊
        - **话题健康**：不要在公共频道讨论不当或引战的话题。🙉
        - **待人友善**：请尊重每一位玩家。🙈
        - **禁止刷屏**：不要在聊天频道重复发送信息。😡
        - **保护隐私**：不要询问他人的个人信息。🕵️‍♂️
        - **形象得体**：不要使用不合适的皮肤或建造不雅的建筑。😳
        - **和谐相处**：不要在公开场合吵架。😭
        - **专注本服**：不要宣传或提及其他服务器。😒

    2. 禁止抢劫，破坏，或者偷窃
        - **不许偷窃**：从别人的箱子里拿东西是不对的！😠
        - **不许破坏**：不要破坏别人辛辛苦苦搭建的方块。😡
        - **爱护建筑**：严禁以任何方式损害他人的建筑。😤
        > **重要提示**：请记住，不是你的东西不要碰！服务器提供了锁箱和领地系统来保护你的财产，请务必使用它们。如果没有做好保护措施导致物品丢失，我们可能无法帮你找回哦。

    3. 禁止黑客，作弊，利用漏洞，或者滥用服务器😡
        - **公平游戏**：不要使用任何作弊模组、宏或游戏漏洞来获得不公平的优势，例如透视、寻路、雷达等。👀
        - **报告漏洞**：发现任何游戏或服务器的漏洞，请立即停止利用并向工作人员报告。🙋‍♀️
        - **爱护服务器**：我们是多人生存服务器，请不要建造卡服机器或进行任何可能导致服务器延迟、崩溃的行为。如果你的建筑影响了服务器性能，工作人员可能会联系你进行调整。😿
        - **合理自动化**：允许使用自动点击器等工具进行自动化操作，但前提是你必须在电脑前，并且能够随时回应工作人员。挂机状态下不允许自动化。同时，像杀戮光环、超快连点这类破坏平衡的功能是绝对禁止的！❌

    4. 禁止欺骗别人哦🚫
        - **诚信交易**：严禁任何形式的欺诈行为，比如利用商店标价错误、收钱不办事、坑骗新手等。😈
        - **店主责任**：商店主人有责任维护好自己的商店价格。如果长时间不上线，记得把商店关掉哦。🔒
        - **合理宣传**：在广告中使用“最便宜”、“最好”等夸张词汇属于宣传手段，不算欺骗。但玩家也需要自己判断物品的价值哦。💰
        - **禁止赌博**：不可以开设抽奖、赌博等可能无法兑现的活动。🎲
        - **尊重隐私**：未经主人同意，不可以泄露他人基地或前哨的位置。🏠
        - **权限说明**：出售建筑（如旅馆房间）的使用权是可以的，但必须向买家明确说明，他们购买的只是进入权限，而不是整个领地的所有权。🏷️

    5. 不要伤害其他玩家哦👊
        - **禁止PVP**：在生存世界中，禁止通过任何方式（直接或间接）伤害其他玩家，包括设置陷阱。😭
        > **例外**：在PVP竞技场或官方举办的活动中，PVP是允许的。🥊

    6. 不要做一些违法的事情哦🔒
        - **正版账号**：请使用从官方渠道购买的正版Minecraft账号。使用盗版或黑卡账号会导致关联IP和账号被封禁。如果你的账号被盗，请立即联系工作人员暂时封禁以保护你的财产。😇
        - **禁止RMT**：严禁使用真实货币（RMT）交易游戏内的任何物品、金钱或服务。这包括用真实物品换游戏币，或用真钱买游戏道具等。😠
        > **允许的交易**：免费赠送真实物品、用游戏币换取社区签名/头像等艺术创作。🎨
