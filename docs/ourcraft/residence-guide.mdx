# 工艺小镇 领地说明

## 开始建设一个领地吧！

- **来建设属于您的领地！**🏠

    - 领地是一种可以保护您的领地不被别人破坏的神奇功能。您可以自己创建和管理您的领地，让您的建筑更安全更美丽🏠
    - **限制**🚫
        - 要查看您能创建的领地的大小和数量，输入 `/res limits` 🔍
        - 一般来说，您可以创建一个最大为100x100x256的领地，或者多个小一点的领地。😊
    - **费用**💰
        - 注意哦，创建领地需要花费游戏里的OurCraft Token💰。每保护一个方块需要花费2.4 OCT。比如说，如果您保护一个7x7x4的区域，就需要花费470OCT。😥
        - 您可以通过完成任务，出售物品，参与活动等方式来获取OCT哦！😉
    - **准备**🛠
        - 要选择您想要保护的区域，您需要一把木斧🪓。如果您没有，可以现做一把。😄
        - 领地是按照立方体的形状来保护的，所以您要找到您想要覆盖的区域，然后用木斧选定两个对角的方块。👉
    - **选择领地区域**🌐
        - 拿着您的木斧（参见上面的准备部分），左键点击您想要保护区域的左下角的方块，然后走到对面，右键点击右上角的方块。这样就可以选定一个立方体区域了。👏
        - 当您选择第一个点时，系统会显示出主要选择点的坐标。选择第二个点时，系统会显示出次要选择点的坐标。👀
    - **创建领地**🎉
        - 在您选择好领地区域后，输入以下命令：`/res create 领地名字`（其中领地名字是您自己取的一个名字）。这样就可以创建好您的领地了🎉
        - 恭喜您，现在您可以享受您的安全和舒适的领地了😍

## 权限和标志们

您可以给您的领地设置不同的权限和标志，让您有更多的自由和控制👍

### 设置方法

- **如何设置领地的标志呢？**🚩

    - 标志是一种可以让您控制您的领地的权限和功能的方式，您可以给不同的人或组设置不同的标志哦！😊
    
    - 设置方法
        - `/res set <领地名字> [标志] [true/false/remove]` - 给领地设置标志。👉
        - `/res gset <领地名字> [组名] [标志] [true/false/remove]` - 给不同的组设置标志。👥
        - `/res pset <领地名字> [玩家名字] [标志] [true/false/remove]` - 给不同的玩家设置标志。👤
        - `/res lset <领地名字> [黑名单/白名单] [材料]` - 把一个材料加入或移出领地的黑名单/白名单。🔳
        - `/res lset <领地名字> info` - 查看领地的黑名单/白名单设置。🔍
        - `/res mirror [完整源名字] [目标名字]` - 让目标空间拥有和源空间一样的标志。🔁

    - 标志说明
        - `move` - true表示允许移动，false表示禁止移动。🚶
        - `build` - true表示允许建造，false表示禁止建造。🏗
        - `use` - true表示允许使用按钮，拉杆，门，和工作台，false表示禁止使用。🔘
        - `container` - true表示允许使用箱子，熔炉，和发射器，false表示禁止使用。📦
        - `pvp` - false，领地不允许玩家之间互相攻击。⚔️
        - `ignite` - false，领地不允许玩家点燃火焰或引爆TNT。🔥
        - `subzone` - true表示允许创建子区域，false表示禁止创建子区域。🌐
        - `firespread` - false，领地不允许火焰蔓延或烧毁方块。🚫
        - `damage` - false，领地不允许玩家受到任何伤害。❤️
        - `tnt` - false，领地不允许TNT爆炸或破坏方块。💥
        - `creeper` - false，领地不允许苦力怕爆炸或破坏方块。💣
        - `monsters` - false，领地不允许生成任何敌对生物。😈
        - `flow` - true表示允许液体流动，false表示禁止液体流动。💧
        - `tp` - false，领地不允许玩家在领地内传送或使用末影珍珠。🚫

    您可以根据您的喜好和需求来设置不同的标志，让您的领地更加完美😍

## 子区域

- **子区域是什么**🌐

    - 您可以把一个领地区域划分成更小的子区域，来管理更细致的权限。这样您就可以创建一个大的领地区域，然后把它分成多个子区域，给每个用户他们自己的子区域和建造权限。😊
    - **创建子区域**🎉
        - 要创建一个子区域，您用木斧选择一个区域，就像您选择领地一样。👉
        - 选择好后，输入以下命令：`/res subzone <领地名字> <子区域名字>` 📝
        - 注意哦，子区域必须包含在领地区域里面，而且不能和其他子区域重叠。🚫
        - 子区域的名字会是 领地名字.子区域名字 😄
        - 另外一个子区域的特点是，它们可以嵌套。比如说，您可以在一个子区域里面再创建一个子区域。那么名字就会是 领地名字.子区域名字.下级子区域名字 😍
    - 关于如何给子区域设置权限，请参见上面的权限和标志部分（只要把<领地名字>换成<领地名字>.<子区域名字>就行了）。🚩

## 市场

- **市场**🛒

    - 您可以成为一个房东，把您的领地里的子区域出租给租客。他们会出现在市场里，您可以收取租金。💰
    - **出租方法**🏘
        - `/res subzone <领地名字> [子区域名字]` - 在您的领地里创建一个子区域。您必须是领地的主人才能这样做。👉
        - `/res market rentable [主领地名字].[子区域名字] [租金] [# of days] [repeat: true/false]` - 把一个子区域放到市场上出租。👏
        - `/res market [buy/info] [领地]` - 购买 / 查看一个出售的领地的信息。👀
        - `/res market sell [领地] [金额]` - 把一个领地放到市场上出售。💸
        - `/res market unsell [领地]` - 把一个领地从市场上撤下。🚫
        - `/res lease [renew/cost] [领地]` - 续租 / 显示续租一个领地的费用。💵
        - `/res market list` - 查看所有可购买或出租的领地。📜
        - `/Balance` - 查看您当前的金额。💰

    您可以通过市场来赚取更多的钱💰，也可以找到适合自己的地产🏡

## 消息

- **消息**📩

    - 您可以控制别人进入您的领地时看到的消息。😊
    - 在修改消息时，您可以用 `%player` 来显示玩家的名字，用 `%residence` 来显示领地的名字，用 `%owner` 来显示主人的名字。😄
    - 设置消息的命令是：📝
        - `/res message [领地] [enter/leave] [消息]` 📝
        - 比如说，如果您有一个叫做“我的房子”的领地，您可以设置一个这样的消息：👉
        - `/res message 我的房子 enter 您好 %player - 欢迎来到 %residence - 由 %owner 创建` 👏
        - 这样，别人进入您的领地时就会看到这样的消息：👀
        - 您好 李华 - 欢迎来到 我的房子 - 由 本喵们 创建 😍

    您可以用这个功能来给您的领地增加一些个性和趣味😉
