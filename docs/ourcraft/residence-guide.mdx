# 工艺小镇 领地说明

## 🏠 开始建设一个领地吧！

领地是一个神奇的保护罩，可以保护你的心血之作不被破坏或窃取。快来学习如何创建和管理属于你自己的安全小天地吧！

### 1. 了解限制与费用

#### **限制** 🚫
每位玩家可以创建的领地数量和大小是有限的。通常，你可以创建一个最大 **100x100x256** 的领地，或者多个小领地。
- 输入 `/res limits` 查看你当前的具体配额。

#### **费用** 💰
保护领地需要花费游戏币（OurCraft Token, OCT）。
- **价格**：每保护一个方块需要 **2.4 OCT**。
- **举个例子**：保护一个 `7x7x4` 的区域需要 `7 * 7 * 4 * 2.4 = 470.4 OCT`。
- **赚钱**：你可以通过完成任务、出售物品、参与活动等方式来赚取OCT哦！😉

### 2. 圈地与创建

#### **准备工具** 🛠️
首先，你需要一把**木锄**来圈定领地的范围。如果背包里没有，就地取材做一把吧！

#### **选择区域** 🌐
领地是一个立体的保护区域。你需要用木锄选定这个立方体的两个对角。
1.  手持木锄，**左键**点击一个点作为起点（比如区域的左下角）。
2.  走到对角，**右键**点击另一个点作为终点（比如区域的右上角）。
> 聊天框会显示你选择的坐标，方便你确认范围。

#### **创建领地** 🎉
选好范围后，输入以下指令来创建领地：
```
/res create <你的领地名>
```
> 领地名可以自定义，比如 `MySweetHome`。

恭喜！你的领地现在已经受到保护啦！

## 🚩 权限标志（Flags）

你可以通过设置“标志”来精细地管理领地权限，决定谁可以在你的领地里做什么。

### 设置指令

- **为整个领地设置**：`/res set <领地名> <标志> [true/false/remove]`
- **为特定玩家设置**：`/res pset <领地名> <玩家ID> <标志> [true/false/remove]`
- **为特定组设置**：`/res gset <领地名> <组名> <标志> [true/false/remove]`

### 常用标志说明

| 标志 (Flag) | `true` (允许) | `false` (禁止) |
| :---------- | :--------------------------------- | :----------------------------------- |
| `move` 🚶   | 玩家可以进入和在领地内移动         | 玩家无法进入领地                     |
| `build` 🏗️  | 玩家可以放置/破坏方块              | 玩家无法放置/破坏方块                |
| `use` 🔘    | 玩家可以使用按钮、拉杆、门等       | 玩家无法使用这些物品                 |
| `container` 📦 | 玩家可以打开箱子、熔炉、发射器等   | 玩家无法打开这些容器                 |
| `pvp` ⚔️   | 允许玩家间互相攻击                 | **禁止**玩家间互相攻击（默认）       |
| `ignite` 🔥 | 允许点火和引爆TNT                  | **禁止**点火和引爆TNT（默认）        |
| `flow` 💧   | 允许水和岩浆流动                   | 禁止液体流动                         |
| `tp` 🌀     | 允许使用 `/tp` 或末影珍珠进入领地  | **禁止**通过传送进入领地（默认）     |
| `damage` ❤️ | 允许生物和环境对玩家造成伤害       | **禁止**玩家在领地内受到任何伤害（默认） |
| `monsters` 😈 | 允许敌对生物在领地内生成           | **禁止**敌对生物生成（默认）         |
| `creeper` 💣 | 允许苦力怕爆炸破坏方块             | **禁止**苦力怕爆炸破坏方块（默认）   |
| `tnt` 💥    | 允许TNT爆炸破坏方块                | **禁止**TNT爆炸破坏方块（默认）      |

## 🗺️ 子区域 (Subzone)

想在你的大领地里给朋友划分一小块专属空间吗？子区域功能可以帮你实现！

### 创建子区域
1.  像创建主领地一样，用木锄在**主领地内部**选择一块区域。
2.  输入指令：`/res subzone <主领地名> <子区域名>`

子区域创建成功！它的完整名称是 `主领地名.子区域名`。你甚至可以在子区域里再创建子区域（嵌套），名称会像这样：`主领地名.子区域名.下级子区域名`。

### 管理子区域
给子区域设置权限的方法和主领地一样，只需把指令中的 `<领地名>` 换成子区域的完整名称即可。

## 🛒 领地市场

你可以成为一名“房东”，将你的领地或子区域放到市场上出售或出租来赚取OCT！

### 出售/出租你的领地
- **出售领地**：`/res market sell <领地名> <价格>`
- **出租子区域**：
  1.  先创建一个子区域（见上文）。
  2.  使用指令将其挂牌出租：`/res market rentable <领地名.子区域名> <每次租金> <天数> [是否可续租:true/false]`
- **下架**：`/res market unsell <领地名>`

### 购买/租赁市场上的领地
- **查看市场**：`/res market list`
- **查看详情**：`/res market info <领地名>`
- **购买**：`/res market buy <领地名>`
- **续租**：`/res lease renew <领地名>`

## 💬 自定义领地消息

想让玩家进入你的领地时看到独特的欢迎语吗？你可以自定义进入（enter）和离开（leave）时的消息。

### 设置指令
`/res message <领地名> [enter/leave] <你的消息>`

在消息中，你可以使用一些有趣的变量：
- `%player` - 显示玩家的名字
- `%residence` - 显示领地的名字
- `%owner` - 显示领地主人的名字

**举个例子：**
```
/res message MySweetHome enter 欢迎 %player 来到 %residence！这里是 %owner 的温馨小屋~
```
当其他玩家进入时，他们就会看到这条可爱的欢迎消息啦！
