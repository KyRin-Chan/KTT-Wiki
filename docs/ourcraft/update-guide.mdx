# 工艺小镇 升级说明

## 呀，版本不对？教你轻松搞定！

如果你被传送到了这里，很可能是因为你的游戏版本和服务器不匹配哦。

工艺小镇（OurCraft Network）目前使用的版本是 **1.20.4**。

别担心，跟着下面的简单几步，马上就能加入我们啦！😉

---

### Minecraft 启动器设置指南 🚀

1.  打开你的 Minecraft 启动器。

2.  在启动器顶部，找到并点击 **「安装」** 选项卡。

3.  点击 **「+ 新建安装」** 按钮，或者直接编辑你常用的那个配置。

4.  在设置页面里，找到 **「版本」** 下拉菜单。

5.  从列表中选择 **`release 1.20.4`** 这个版本。

6.  点击右下角的绿色 **「创建」** 或 **「保存」** 按钮。

7.  回到主界面（「Java版」选项卡），在绿色的「开始游戏」按钮旁边，选择你刚刚设置好的 `1.20.4` 版本配置。

8.  点击 **「开始游戏」**，然后就可以顺利连接到工艺小镇啦！👏
