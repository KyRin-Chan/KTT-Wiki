# 工艺小镇 升级说明

## 开始升级您的版本吧！

- **如何升级到最新的MC版本？**🎮

    工艺小镇网络（OurCraft Network）目前使用的是1.19.4版本，而最新的我的世界（Minecraft）版本是1.20。

    您被重定向到这个页面是因为你试图用一个比OCN支持的版本更旧的我的世界版本来连接我们的服务器。

    如果您想开始使用我的世界1.20版本，请按照以下步骤操作哦！😉

- **新版启动器**🚀
    1) 打开我的世界启动器。
    2) 点击启动器顶部附近的安装按钮。
    3) 点击“+ 新建”按钮，或者从列表中点击你使用的配置文件。你会看到该配置文件的设置。
    4) 不管你在上一步做了什么，在启动器里显示“版本”的地方，拉下下拉框。点击显示为1.20版本的地方，让启动器使用OCN目前使用的版本。
    5) 按下底部的绿色保存按钮。回到游玩标签（安装按钮左边）。
    6) 你可以从游玩按钮旁边的下拉菜单中选择你的配置文件。你会注意到“示例配置文件”使用了1.20版本，这是最适合OCN的。点击玩耍，然后你就可以连接到OCN啦！👏
